# Hierarchical Bayesian Quantile Regression - SABER 11

Estimation of the effect of socioeconomic covariates on the **80th percentile** of the overall SABER 11 exam score (Colombia, 2023-2), using hierarchical Bayesian quantile regression with a student → school → municipality structure.

## Method

An **EM with INLA** algorithm is used, based on the representation of the Asymmetric Laplace Distribution (ALD) as a normal-exponential mixture (Kozumi & Kobayashi, 2011; Yue & Rue, 2011):

1. **E-step**: given the latent weights w_i, INLA fits a weighted hierarchical Gaussian model (~20s with 415K obs)
2. **M-step**: given the parameters, the w_i are updated using a closed-form expression from the GIG distribution
3. Converges in ~20 iterations (~10 min total)

See [metodologia.md](metodologia.md) for the full description of the algorithm, model, and references.

## Data

The dataset `SB11_20232.TXT` contains the SABER 11 exam results from the second semester of 2023, published by ICFES. It is not included in the repository due to its size (~450MB). It must be placed in the project root.

**Source**: [ICFES - Open Data](https://www.icfes.gov.co/)

## Covariates

| Variable | Description |
|----------|-------------|
| `estrato` | Socioeconomic stratum (1–6) |
| `educ_madre` | Mother's education (None, Primary, Secondary, Technical, Professional, Postgraduate) |
| `educ_padre` | Father's education (same levels) |
| `internet` | Internet access at home (Yes/No) |
| `horas` | Works during the week (Works/Does not work) |
| `naturaleza` | School type (Public/Private) |

## Random Effects

- **Municipality** (`muni_idx`): random intercept by school municipality
- **School** (`cole_idx`): random intercept by school

Priors: PC priors with P(sigma > 50) = 0.01.

## Contrasts

Two versions of the model are fitted:
- **Sum-to-zero**: each coefficient is the deviation from the global mean (Bayesian ANOVA)
- **Treatment**: each coefficient is the difference relative to the reference category
